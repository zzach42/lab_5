# Задание 1. Произведение нечетных отрицательных чисел (бинарный файл)

Для выполнения задания бинарный файл заполняется случайными целыми числами.
Далее последовательно выполняются следующие действия:

Заполнение файла случайными числами

Вывод содержимого файла

Вызов метода, который:

читает файл

проходит по всем элементам в цикле

с помощью оператора ветвления отбирает нечетные отрицательные числа

вычисляет их произведение

Пример 1:
Случайная генерация:

-12 -29 30 -24 37 -48 -30 -19 23 17 0 -31 39 5 25

Результат:

-17081

Пример 2:
Случайная генерация:

29 17 43 1 -8 31 -18 -42 -47 -28 3 19 -41

Результат:

1927

Результаты вычислений являются корректными





# Задание 2. Работа с матрицей (бинарный файл)

Пользователь вводит размер матрицы n.
Из бинарного файла создаётся квадратная матрица n×n. Если элементов файла недостаточно, оставшиеся элементы заполняются нулями.

Далее в отдельном методе:

выполняется проход по каждому столбцу

находятся минимальный и максимальный элементы

они меняются местами с помощью std::swap



# Задание 3. Сумма квадратов элементов (текстовый файл)

Из текстового файла числа считываются построчно.
Для каждого элемента вычисляется квадрат, который добавляется к общей сумме.

Пример 1:
Входные данные:

4
-4
10
8
6
-10

Результат:

317

Пример 2:
Входные данные:

4 3 1 -2 8 -3 4 1

Результат:

104

Оба результата являются корректными.



# Задание 4. Произведение элементов файла

Текстовый файл содержит числа, расположенные по несколько в строке.
Вычисление производится следующим образом:

файл читается построчно

каждая строка разбивается на числа

все элементы перемножаются в цикле



% Задание 5. Копирование строк заданной длины

Необходимо переписать строки длины m из одного текстового файла в другой.

Для этого:

открываются два файла: один на чтение, второй на запись

файл читается построчно

длина строки определяется методом .length()

если длина равна m, строка копируется в новый файл
